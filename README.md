# scmroadwiki

# 一些约定
1. 请为每个一级页面简历一个目录
2. 每个工具的内容尽量都在一个页面里呈现
3. 每个页面包含但不限于：简介（一句话），安装，配置，使用，最佳实践，相关资料
4. 每个页面的格式请遵循 markdown 语法
5. 把工作分成一个个细小的 section，完成一个后及时 pr，以免冲突和浪费时间
6. 高度自我管理、自我认领任务
